## Troubleshooting

### Common Issues

#### Can't Access the Platform?
- Verify your team code is correct (8 characters, case-sensitive)
- Try refreshing the page
- Contact event organizers if login continues to fail

#### AI Not Responding?
- Check your internet connection
- Try switching to a different AI model
- Refresh the page if responses seem stuck

#### Images Not Generating?
- Ensure your prompt follows content guidelines
- Try simpler, more specific prompts
- Wait for current generation to complete before starting another

#### Session Expired?
- Re-enter your team code to restore access
- Previous chat history may be preserved
